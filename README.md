# Weather App

A simple web-based Weather App to display current weather of any city.

## Features
- Search weather by city
- Display temperature and weather description
- User-friendly UI

## Tech Stack
- HTML
- CSS
- JavaScript
- Fetch API

## How to Run
1. Clone this repository
2. Open `index.html` in your browser
3. Replace `YOUR_API_KEY` in the script with your own valid API key

## Disclaimer
This project requires a valid API key to fetch live weather data. Without a proper key, the live data will not display.
